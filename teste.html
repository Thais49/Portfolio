Teste de GIT e GITHUB

Como fazer commit enviar ao github?

1. para listar as configurações do GIT: git config --global --list

2. para clonar repositorio: git clone 'link do repositorio'

3. nao esquecer na primeira vez ira pedir autorização do git

4. salvar o arquivo que esta escrevendo.

5. comando para criar um arquivo: git add nomedoArquivo

6. comando para gravar commit: git commit nomedoArquivo a ''sua mensagem'

7. comando para enviar ao repositorio Github: git push origin main
